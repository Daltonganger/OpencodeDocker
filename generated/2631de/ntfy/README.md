# ntfy Stack

Self-hosted push notification service behind Traefik.

## Deploy

```bash
docker compose up -d
```

## Access

- URL: https://notify.2631.eu

## Configuration

- Config: `config/ntfy/server.yml`
- Edit and restart: `docker compose restart`
